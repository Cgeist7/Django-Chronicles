Skills

Skills are ways to explain what your character can do. Each game system treats skills differently, but most of them tend to both designate a skill and designate a skill value that tells you how good you are at using the skill.

For each Skill, choose a name that describes the skill or talent or choose from a pre-prepared skill list for your game.

You will also need to choose a skill type, which defaults to "Genral". Because this is set up for a GUMSHOE game, we've included an "Investigative" option as well. More options still to come!
